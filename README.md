PathLogger - 通った道を蓄積するシンプルな地図アプリ


📌 概要

※ 本アプリでは、iOS向け無料アプリ「Open GPX Tracker」で取得した .gpx ファイルを記録データとして使用しています。シンプルで扱いやすく、散歩やサイクリングの記録用途に最適です。

このアプリは、散歩やサイクリングなどで通った道を蓄積・可視化するためのシンプルなWebアプリです。

スマートフォンのGPS記録アプリ（例：Open GPX Tracker）で取得した .gpx ファイルをアップロードするだけで、自分が今まで通った道を自動的に地図上に描画します。「いつ、どこを通ったか」ではなく「どれだけの道を通ってきたか」にフォーカスした記録ツールです。


🛠️ 使用技術

・Python 3.x

・Flask（Webアプリケーションフレームワーク）

・folium（地図描画ライブラリ）

・gpxpy（GPXファイル解析用ライブラリ）

・pandas（データ処理）


🚀 機能

.gpxファイルを一括で読み込み、位置情報をCSV形式に変換

・これまで通った全ルートを累積して1枚の地図に描画

・FlaskベースのWebアプリで、スマホやPCからアップロード・閲覧可能

・面倒な操作なしで、自動的に地図を更新して表示


📂 ディレクトリ構成

path-logger/

├── gpx_batch_convert.py   # gpx→csv変換スクリプト

├── .gitignore
└── README.md


🧪 実行方法

🖥️ ターミナル実行モード（開発者・PC利用向け）

1. 必要ライブラリをインストール：

pip install flask folium pandas gpxpy

2. gpx_logs/ フォルダに .gpx ファイルを入れる（例：Open GPX Tracker で記録したファイル）

3. 以下を順番に実行：

python3 run_all.py       # gpx→csv変換 & 地図HTML生成
python3 app.py           # Flaskアプリ起動

4. ブラウザで以下にアクセス：

http://localhost:5000/

📱 ブラウザ実行モード（ユーザー向け簡易操作）

1. 上記と同様に python3 app.py を起動すると、Webアプリとして動作します。

2. スマホやPCからブラウザで http://localhost:5000/ にアクセス

3. .gpx ファイルをブラウザ上でアップロード

4. 通過ルートが地図上に表示されます

※ 実際のスマホで使うには、同一Wi-Fi内でホストPCのIPアドレスを指定する必要があります（例：http://192.168.1.5:5000/ など）。


💡 このアプリを作った理由（開発背景）

このアプリを作ろうと思ったきっかけは、私の趣味である散歩に対する「自分なりの楽しみ方」を可視化したいと感じたことです。

私は、歩いたことのある道を増やしていくことで、頭の中に地図が少しずつ出来上がっていく感覚にとても楽しさと満足感を覚えます。自分は道の覚えが良いため、この感覚を自然に頭の中で味わっていますが、「この楽しさを他の人にも共有したい」「散歩のこういう楽しみ方って面白くない？」という提案を形にしたくて、このアプリを開発しました。

既存の散歩関連のアプリは「いつ」「何km」といった数値的な記録や、ポケモンGOやモンハンNOWのように散歩に付加価値を与えることを主目的としたものが中心です。
一方で、本アプリは「通った道そのものを蓄積する」という体験に特化し、できるだけシンプルな設計で構築することで、散歩のより本質的な楽しみ方を提供する“ゲーム”とも言える存在を目指しています。
